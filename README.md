# Sales Data Analysis Dashboard

## Overview
This project features an interactive **Sales Dashboard** built in Microsoft Excel. It is designed to provide actionable insights into business performance by analyzing sales trends, product profitability, and customer demographics.

## Dashboard Highlights
The dashboard provides a comprehensive view of business metrics, including:
* **Top Countries by Revenue:** Geographic distribution of sales.
* **Demographic Analysis:** Insights into sales performance by Age Group and Customer Gender.
* **Product Performance:** Identification of top-selling products and category-wise revenue share.
* **Monthly Trends:** Visualizing the relationship between sales volume and profitability over time.

## Key Features
* **Interactive Filtering:** Uses Slicers for dynamic data exploration (by Country, Month, and Product).
* **Data Visualization:** Employs Pivot Charts to effectively communicate complex datasets.
* **KPI Tracking:** Monitors Total Sales and Profit trends to support data-driven decision-making.

## Tools & Techniques
* **Software:** Microsoft Excel
* **Core Functions:** 
    * Pivot Tables & Pivot Charts
    * Slicers for Interactive Filtering
    * Data Cleaning & Formatting
    * Trend Analysis

## How to View
1. Download the Excel file from this repository.
2. Open it in Microsoft Excel.
3. Use the Slicers on the right to filter the data and explore different performance views interactively.
